
# Periodic Table (PySide6)

A lightweight, interactive Periodic Table application built with Python and PySide6. This app provides a visual representation of all 118 elements with detailed chemical properties available at a click.

## Features
- **Complete Dataset:** Includes all 118 elements (Hydrogen to Oganesson).
- **Interactive UI:** Click any element to view its details in a side panel.
- **Detailed Info:** Displays atomic number, electron configuration, discovery info, melting/boiling points, and more.
- **Visual Coding:** Color-coded layout for different element groups with a modern dark theme.

## Installation

Ensure you have Python installed on your system.

1. **Install Dependencies:**
   ```bash
   pip install PySide6

```

2. **Run the Application:**
```bash
python periodic_table.py

```



## File Structure

* `periodic_table.py`: The main application logic and GUI layout.
* `elements.py`: A comprehensive database containing the properties of each element.
